# Piñateria Lizzety Frontend

Frontend para gestion de inventario de Piñateria Lizzety (responsive).


---

## Componentes

El proyecto se ha desarrollado utilizando las siguientes tecnologías:

- **Java JDK 19**
- **Java Swing**
- **Maven**
- **JCalendar** (libreria)
- **IDE Netbeans** (Entorno de Desarrollo)
